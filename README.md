# OpenHouse App 使用说明网站

这是一个纯静态说明网站，用于展示 OpenHouse App 的维护中心、网页维护器、插件源和 Agent 安装流程。

线上地址：

```text
https://jiwuyou.github.io/openhouse-app-guide-site/
```

## 打开方式

直接用浏览器打开：

```text
/root/openhouse-app-guide-site/index.html
```

## 文件

- `index.html`：说明网站页面
- `styles.css`：页面样式
- `assets/maintenance-center.png`：维护中心截图
- `assets/maintenance-dynamic.png`：动态插件阶段截图
